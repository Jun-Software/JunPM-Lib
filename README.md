# JunPM-Lib
The library of JunPM.

使用方法：
> Usage:

例如，包`test`首字母为`t`，
> For example, if the package name is `test` starting with the letter `t`.
则在`/libs/` `t` `/`文件夹内创建`test.json`，
> Create a file named `test.json` inside the `/libs/` `t` `/` folder.
内容为：
> The content should be:
```json
{
    "name": "test",
    "url": "https://example.com/test.dll"
}
```
